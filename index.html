Introduction

Purpose

This style guide provides essential standards and practices for creating consistent training and curriculum materials.

Our material adheres to the guidelines outlined in Pega Technical Documentation Guidebook and Pega Brand Guidelines unless otherwise stated.

Naming convention

Structure and format

Zero padding: Always use two digits for each level of the hierarchy (e.g., 01, 02, ..., 10).

Hierarchical naming: Each subfolder inherits and extends the parent folder's number (e.g., 0101 is a subfolder of 01).

No punctuation: Do not use dots, underscores, hyphens, or spaces in folder names.

Scalable: This format allows for unlimited expansion without renaming existing folders.

Examples

01 Parent folder 1
    0101 Child folder 1
        010101 Nested folder 1
        010102 Nested folder 2

02 Parent folder 2
    0201 Child folder 1
        020101 Nested folder 1
        020102 Nested folder 2

VISUALS

Whitespace

  • Whitespace is the empty space between elements that enhances readability and balances layout.
  • Proper use of whitespace creates a clean and organized design.
  • Insufficient whitespace leads to cluttered content, making it harder to read and process.

Alignment

Alignment is arranging elements along a common edge or center to create visual order and enhance readability.

Not Recommended: Misaligned elements result in a disorganized, hard-to-follow design.

Recommended: Consistent alignment of text and images creates an organized, easy-to-read layout.
